
**Tic-Tac-Toe**

This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. It’s designed for two players, allowing you to play against a friend while keeping track of turns and results in real time. The interface is clean and interactive, making the game easy and enjoyable to use.

**Features**
The game includes a responsive and interactive layout, clearly shows whose turn it is, and automatically detects when a player wins or when the game ends in a draw. There’s also a reset option so you can quickly start a new round without refreshing the page.

**How to Play**
Player X always starts the game. Players take turns clicking on empty tiles to place their mark. The game switches turns automatically after each move. The first player to align three marks in a row, column, or diagonal wins. If all tiles are filled without a winner, the game ends in a draw. You can use the reset button to start over at any time.

**Technologies Used**
The structure is built with HTML, styled using CSS, and powered by JavaScript for handling the game logic and interactions.

**Why Play**
Tic-Tac-Toe is a quick and engaging game that encourages strategic thinking. It’s a great way to take a short break and enjoy a friendly challenge.
